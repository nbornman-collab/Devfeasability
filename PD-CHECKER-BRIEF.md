# DevFeasibility — PD Checker Product Brief

New product strand. High volume, lower price point. Separate from the developer-facing tool.

## What it is
A Permitted Development rights checker for residential properties in England. User enters an address, gets back a clear breakdown of what they can build without planning permission.

## Target customers
- Homeowners wanting to know what's possible before calling an architect
- Small builders and contractors doing resi extensions
- Residential architects doing early-stage client advice
- Planning consultants handling PD / Prior Approval work

## What needs building
- Article 4 direction layer (per-LPA, some available via APIs, some need scraping)
- PD rules engine (dimension logic per Class - can be coded, LPA-agnostic for national rules)
- Clean resi-facing UI (different from developer tool - simpler, consumer-grade)
- Paywall / report delivery

## Pricing model
- Per report: £99 one-off
- Professional sub: £49/mo (unlimited checks, for consultants/builders doing volume)

## Why it works
- TAM is massive - hundreds of thousands of homeowners per year asking this question
- Acquisition via SEO ("can I build a rear extension without planning permission London")
- Low marginal cost once built
- Some PD users graduate into full developer product (natural upsell)

## Priority
Build after core developer product has paying customers. Could be Sprint 3 or a parallel lightweight MVP.
