How it works
Borough to blueprint in four steps

Each tier is a decision gate. Stop when you have enough. Go deeper when you need to.

T0
Find the opportunity
Borough-wide ranking by 6 development intelligence factors. Identify which sites are worth looking at before committing time.
T1
Qualify the site
Planning flags, ownership, heritage risk, height context, transport connectivity. Enough to decide if a site warrants a full appraisal.
T2
Run the numbers
Three massing scenarios, live financial model, IRR on equity, return curve. Professional development appraisal, all assumptions visible.
T3
Present the case
Investment-grade PDF. Architectural recommendation, financial conclusions, heritage strategy. Ready for board, funder, or JV partner.
